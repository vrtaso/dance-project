01

Описание проекта

Сайт посвящён теме танца как способа самовыражения. Проект состоит из трёх страниц: главная с текстовым рассказом о танце, страница направлений танца и страница со статистикой популярности стилей по возрастным группам.

Сайт выполнен в едином тёмном эстетическом стиле — тёмный фон, бежевые акценты, элегантная типографика. На всех страницах одинаковые шапка (хедер), навигация и визуальный язык.

Главная страница

Рассказ о танце как искусстве и способе выразить чувства. Три текстово-визуальных блока с изображениями и карточки «Почему танец важен?»

index.html

Направления

Слайдер с карточками восьми стилей танца (балет, бальные, jazz, contemporary, hip-hop, breakdance, народные). Секция «О направлениях».

directions.html

Статистика

Визуализация популярности танцевальных направлений по возрастным группам: графики (столбцы/полосы) и сводная таблица с выводами.

statistics.html

Цветовая система

Все цвета заданы через CSS-переменные (:root). Это позволяет легко поменять всю палитру в одном месте — изменив значение переменной, цвет поменяется сразу на всём сайте.

CSS — переменные 
:root {
--bg-color: #1a1a1a;    /* тёмно-серый фон */
--text-main: #e0e0e0;  /* светло-серый текст */
--accent:   #d4c5b0;   /* бежевый акцент */
--white:    #ffffff;   /* белый для заголовков */
}

Как используется акцентный цвет

Бежевый #d4c5b0 применяется для: активных пунктов навигации, подчёркивания ключевого слова в заголовке (outline-эффектом), линии-разделителя в логотипе, иконок карточек, рамок элементов, подписей и меток.

Внешние библиотеки

Сайт подключает внешние библиотеки через CDN (Content Delivery Network) — готовые серверы, которые хранят популярные файлы и отдают их быстро. Ничего не нужно скачивать или устанавливать — всё подключается через тег <script src="...">.

GSAP (GreenSock)
v3.12.2 — gsap.min.js

Главная библиотека анимаций. Управляет всеми движениями на сайте: появление заголовков, карточек, параллакс-эффект на картинках.

cdnjs.cloudflare.com/ajax/libs/gsap/3.12.2/gsap.min.js
GSAP ScrollTrigger
v3.12.2 — плагин к GSAP

Плагин для GSAP, который запускает анимации при прокрутке страницы. Без него элементы бы анимировались сразу, а не при появлении в окне.

clamp() — адаптивный шрифт
index.html (заголовок)

Функция CSS clamp(min, preferred, max) автоматически подбирает размер шрифта между минимумом и максимумом в зависимости от ширины экрана. Не нужно писать медиа-запросы для каждого брейкпоинта.

.hero-title {
font-size: clamp(2.5rem, 6vw, 5rem);
/* мин: 2.5rem | предпочт: 6% ширины | макс: 5rem */
}

01
Мобильное меню (бургер)

Интерактивность

Находит кнопку-гамбургер и меню по их id. При клике на кнопку добавляет/убирает класс active — это включает CSS-анимацию крестика и показывает/прячет меню. При клике на любую ссылку в меню — закрывает его.

mobileMenuBtn.addEventListener('click', () => {
mobileMenuBtn.classList.toggle('active');
navMenu.classList.toggle('active');
});
document.querySelectorAll('.nav-menu a').forEach(link => {
link.addEventListener('click', () => {
mobileMenuBtn.classList.remove('active');
navMenu.classList.remove('active');
});
});
02
Анимация при загрузке страницы (GSAP Timeline)
Все 3 страницы
GSAP

Запускается когда страница полностью загрузилась (window.addEventListener('load')). Использует GSAP Timeline — это как режиссёрский сценарий: сначала появляется шапка, затем линия логотипа, потом слова заголовка вылетают снизу одно за другим (stagger — задержка между словами).

window.addEventListener('load', () => {
const tl = gsap.timeline();
tl.to(".header", { opacity: 1, duration: 0.25 })
.to(".logo-line", { scaleX: 1, duration: 0.25 }, "-=0.2")
.to(".hero-title .word span", {
y: 0,          // вернуть на место из -100%
duration: 0.4,
stagger: 0.05, // каждое слово — 50мс позже
ease: "power4.out"
}, "-=0.15");
});
03
Анимация при прокрутке (ScrollTrigger)
Все 3 страницы
GSAP

Все элементы с классом .reveal-text появляются плавно, когда попадают в зону видимости экрана. ScrollTrigger отслеживает позицию каждого блока и запускает анимацию в нужный момент.

gsap.registerPlugin(ScrollTrigger);
document.querySelectorAll('.reveal-text').forEach(block => {
gsap.from(block, {
scrollTrigger: {
trigger: block,
start: "top 80%", // когда верх блока = 80% высоты экрана
},
y: 30,       // начать с 30px ниже
opacity: 0,  // начать прозрачным
duration: 0.4,
ease: "power3.out"
});
});

Уникальные скрипты страниц

04
Параллакс-эффект на картинках
index.html
GSAP

При прокрутке картинки слегка «плывут» с разной скоростью — это создаёт глубину (параллакс). Реализовано через scrub: 0.5 — анимация «прилипает» к скроллу с небольшим запаздыванием.

document.querySelectorAll('.reveal-img').forEach(wrapper => {
gsap.to(wrapper.querySelector('img'), {
scrollTrigger: {
trigger: wrapper,
start: "top 85%",
end: "bottom 60%",
scrub: 0.5 // «прилипание» к прокрутке
},
scale: 1,
y: 0
});
});
05
Слайдер карточек (directions.html)
directions.html — самый сложный скрипт
Сложный

Полностью самописный слайдер без готовых библиотек. Работает так:

1. initSlider() — считает ширину карточки и контейнера, определяет сколько карточек видно, создаёт нужное количество точек-навигаторов.
2. updateSlider() — сдвигает «трек» через translateX, обновляет состояние кнопок и активную точку.
3. goToSlide(index) — прыжок к конкретному слайду по клику на точку.
4. При изменении размера окна — пересчитывает всё заново (debounce 250мс). 

function updateSlider() {
const offset = currentIndex * (itemWidth + gap);
track.style.transform = `translateX(-${offset}px)`;
dots.forEach((dot, index) => {
dot.classList.toggle('active', index === currentIndex);
});
prevBtn.disabled = currentIndex === 0;
nextBtn.disabled = currentIndex >= maxIndex;
}

Анимация выводов списком
statistics.html
GSAP

Пункты выводов в конце страницы статистики появляются не все сразу, а по очереди, выезжая слева — эффект «последовательного перечисления». Создаётся параметром stagger: 0.05.

gsap.from(".conclusions li", {
scrollTrigger: { trigger: ".conclusions", start: "top 80%" },
x: -30,       // начать 30px левее
opacity: 0,
duration: 0.4,
stagger: 0.05, // каждый пункт — 50мс позже
ease: "power3.out"
});

Глоссарий терминов

Основные технические понятия и технологии, использованные в проекте, простыми словами.

Инфраструктура

CDN

Content Delivery Network — сеть серверов по всему миру. Позволяет загружать файлы (скрипты, шрифты) не с моего компьютера, а с ближайшего к пользователю мощного сервера, что ускоряет работу сайта.

Анимация

GSAP

GreenSock Animation Platform — профессиональный «движок» для создания анимаций в вебе. Позволяет делать сложные движения элементов плавными и контролируемыми.

Логика

DOM

Document Object Model — представление страницы в виде «дерева» объектов. JavaScript использует его, чтобы находить элементы (например, кнопку меню) и менять их свойства.

Дизайн

Flexbox / Grid

Современные технологии CSS для расположения элементов. Flexbox идеален для строк и меню, а Grid — для сложных сеток и макетов страниц.

Эффект

Glassmorphism

Стиль «матового стекла». Создается с помощью backdrop-filter: blur(), когда фон за элементом размывается, создавая ощущение глубины.

Текст

Web Fonts

Шрифты, которые загружаются из интернета (например, Google Fonts). Они позволяют сайту выглядеть одинаково красиво на всех устройствах, даже если этих шрифтов нет в системе пользователя.

Метод

Stagger

Эффект «каскадного» появления элементов. Когда группа объектов (например, слова в заголовке) анимируется не одновременно, а с небольшой задержкой друг за другом.

Адаптив

Media Queries

«Медиа-запросы» в CSS. Позволяют сайту менять свой вид (цвета, размеры, скрывать блоки) в зависимости от размера экрана — смартфона, планшета или монитора.

Авторство

✦ О помощи видеоуроков в проекте

Скрипты на JavaScript (анимации, слайдер, мобильное меню) были написаны с помощью видеоуроков, поскольку автор не изучала программирование. Это честный и современный подход — видеоуроки выступали инструментом реализации, как фотоаппарат для фотографа.

Весь остальной контент — текст, структура, концепция, дизайн-решения, выбор изображений, тема и её раскрытие — написаны и созданы автором самостоятельно.

Написано автором

Собственная работа

Все тексты на страницах, тема и её раскрытие, концепция сайта, структура страниц, HTML-разметка, CSS-стили, подбор фотографий, статистические данные и выводы.

При помощи видеоуроков

JavaScript-скрипты

Анимации GSAP (загрузка страницы, появление при скролле, параллакс), слайдер направлений с навигацией и адаптивностью, мобильное бургер-меню.
